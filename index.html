Lista de canales Protocolo IRC

Que es IRC:

Internet Relay Chat (IRC) Protocolo 1988, Permite Conectar y comunicarse en salas, chat en tiempo real. (Servers - Salas o Canales - Clientes.)

Protocolos Alternos:

1.Tox
2.Matrix
3.Xmpp (jabber)


Clientes IRC:IRSSI (Terminal) - HexChat (Grafico) - pidgin (Grafico).

*Algunos Comandos para Conectar y Usar: *Info Adicional:
/server Nombre Server
/nick Apodo
/list Muestra Listado Canales
/join #canal Ingresar a canal o sala
/msg NombreUser Enviar Mensaje Privado
/part #canal Abandonar Canal con Mensaje Despedida
/quit Desconectar Server
/whois =User Obtiene info del user
/topic #canal Tema actual del canal
/join #canalnombre Acceder a canal y si no esta creado lo crea
/connect +NameServer Conectar a server
/server list Muestra opciones disponibles
